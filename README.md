# MicoshLogs

MicoshLogs is a class for simple logging system.

## How to use?

1. Include class in your file:
```
include('MicoshLogs.php');
```
2. Instantiate class
```
$MLogs = new MicoshLogs();
```
3. Log what you want!
```
$MLogs->l('This is my log', 'Notice');
```
